## 🎭 5개 시리즈 기획

### 📺 시리즈 1: "8주 완성! Vibe Coding 실전 시리즈"

#### 컨셉
> 8주간 0에서 MVP까지 완성하는 전체 과정을 담은 메인 교육 시리즈

#### 타겟 시청자
- Vibe Academy 수강생
- 독학으로 웹 개발을 배우고 싶은 사람
- MVP 개발에 관심 있는 예비 창업자

#### 영상 구성 (총 16편)

**1-1편: Vibe Coding이란? (15분)**
```markdown
## 스크립트 개요

### 오프닝 (1분)
안녕하세요, 바이칼 시스템즈 대표 [이름]입니다. 
지난 6개월간 40개의 MVP를 직접 만들어보면서 
깨달은 것이 있습니다. 

전통적인 개발 방식으로는 절대 따라갈 수 없는 
속도의 혁신이 일어나고 있다는 것이죠.

오늘부터 8주간, 여러분과 함께 
2주 만에 실제 서비스를 만드는 
'Vibe Coding'의 모든 것을 공유하겠습니다.

### 본론 1: 문제 정의 (3분)
기존 개발 방식의 한계:
- 기획 2개월, 개발 4개월, 테스트 2개월
- 시장 변화 속도를 따라갈 수 없음
- 완벽을 추구하다 기회를 놓침

실제 데이터 제시:
- 스타트업 90%가 1년 내 실패
- 그 중 42%가 "시장 수요 없음"으로 실패
- 빠른 검증의 중요성

### 본론 2: Vibe Coding 솔루션 (8분)
핵심 원칙:
1. AI 프롬프트 기반 개발
2. 최소 기능으로 최대 가치
3. 2주 내 배포 완료
4. 실사용자 피드백 중심

기술 스택 소개:
- Frontend: Vite + React (빠른 개발)
- Backend: Firebase (서버리스)
- 배포: Vercel (자동 배포)
- AI: ChatGPT/Claude (코드 생성)

실제 성공 사례:
- 로또 번호 생성기: 10일 개발, 월 500+ 사용자
- 암호화폐 알림: 8일 개발, 월 800+ 사용자
- 습관 추적기: 14일 개발, 월 300+ 사용자

### 마무리 (3분)
8주 커리큘럼 소개:
- 1-2주차: 기초 및 기획
- 3-4주차: 개발 및 연동
- 5-6주차: 배포 및 최적화
- 7-8주차: 비즈니스화

다음 영상 예고:
"다음 시간에는 실제로 개발환경을 
구축하는 과정을 함께 해보겠습니다."

### CTA (Call to Action)
구독과 알림 설정을 해주시고,
댓글로 만들어보고 싶은 MVP 아이디어를 
공유해주세요!
```

### 💡 시리즈 2: "Prompt Clinic"

#### 컨셉
> 5분 내로 개발 문제를 해결하는 AI 프롬프트 팁 시리즈

#### 영상 형식
- **길이**: 5-8분 (짧고 임팩트 있게)
- **구성**: 문제 제시 → 프롬프트 → 해결책 → 결과
- **스타일**: 빠른 템포, 직관적 편집

#### 에피소드 예시

**EP01: "React 컴포넌트 10초 만에 생성하기"**
```markdown
## 스크립트 (5분)

### 문제 상황 (30초)
"할일 목록 컴포넌트를 만들어야 하는데 
어떻게 시작해야 할지 막막하다면?"

### 프롬프트 소개 (1분)
화면에 실제 프롬프트 표시:

```
당신은 React 전문 개발자입니다.
할일 목록(Todo List) 컴포넌트를 생성해주세요.

기능:
- 할일 추가/삭제
- 완료 체크
- 로컬스토리지 저장

스타일: Tailwind CSS 사용
```

### 실행 과정 (2분)
- ChatGPT에 프롬프트 입력
- 생성된 코드 확인
- VS Code에 복사
- 실행 결과 확인

### 결과 및 팁 (1분 30초)
- 완성된 컴포넌트 시연
- 프롬프트 개선 방법
- 추가 활용 팁

### 마무리 (30초)
"다음 Prompt Clinic에서는 
Firebase 연동을 5분 만에 해보겠습니다!"
```

**EP03: "CSS 애니메이션 자동 생성"**
